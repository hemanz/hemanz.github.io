在 iOS 设备中安装 OpenCV ／ 在 XCode 中导入 OpenCV

环境要求

  • CMake 2.8.8 或更高
  • Xcode 4.2 或更高

获取最新的 OpenCV 代码

从该地址 clone OpenCV repository

在 Mac 也可以直接通过命令行来进行下载

用命令行和 cmake来进行 打包 framework

  1. 建立与 XCode 的链接，让 OpenCV 可以找到编译脚本及编译器。

2.编译 OpenCV framework

如果没有问题，几十分钟后可以在 ~/<my_working_directory>/ios/opencv2.framework中找到 framework 。然后导入到 XCode 中
